# Gene Variant Explorer

An educational web app that helps students and families explore
gene variants using real ClinVar data and short summaries from PubMed.

### Features
- Search by gene (e.g. BRCA1, TP53)
- View known variants and associated conditions
- One-sentence summaries from peer-reviewed research
- No personal data collected

### Data Sources
- ClinVar (NIH)
- PubMed abstracts via NCBI E-utilities

⚠️ Educational use only. Not medical advice, please consult doctor.
